# Flame Recorder

**Flame Recorder** — это легкий, безопасный и ориентированный на конфиденциальность диктофон для Android (minSdk 31+). Благодаря бесплатной бессерверной архитектуре Cloudflare Workers приложение обеспечивает высокопроизводительное распознавание речи (STT) и суммаризацию (LLM) на базе Google Gemini, при этом API-ключи разработчика остаются на 100% защищенными и скрытыми от клиента.

* **Ультралегкий вес:** Размер готового APK составляет менее **10 МБ** благодаря оптимизации R8 и разделению по ABI.
* **Бесплатные STT и суммаризация:** Транскрибирование и аннотирование аудио через безопасный прокси Cloudflare Worker с использованием бесплатного тарифа Gemini Multimodal API.
* **Плавная осциллограмма:** Анимация волны, синхронизированная по VSYNC, с динамическим автовыравниванием пикового усиления для форматов M4A, MP3, WAV и 3GP.
* **Изолированное хранилище (Scoped Storage):** Работа через Storage Access Framework (SAF), что позволяет настраивать пользовательские директории для сохранения.
* **Нативный шеринг:** Безопасная отправка файлов в один клик через `FileProvider` в WeChat, QQ, WhatsApp и Telegram.
